== Description ==
= Plugin All Info =
This plugin provides a simple way to display information about any plugin hosted on the WordPress Plugin Directory. It is also useful for all those who want to display plugin details in their blog posts or pages to remain up to date.

[See live demo](http://www.wpshopee.com/plugin-all-info/)

== Installation ==
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Plugin all info** and click \"*Install now*\"
1. Alternatively, download the plugin and upload the contents of 'plugin-all-info.zip' to your plugins directory, which usually is '/wp-content/plugins/'.
1. Activate the plugin
